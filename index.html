02

Hydroponics AI

STEAM Grant
GitHub

An intelligent hydroponic farming system applying AI, Reinforcement Learning (RL), and IoT to automate nutrient delivery while optimizing energy consumption, plant growth, and nutrient balance. The system continuously learns from sensor data to control environmental conditions and predict yield outcomes — enabling soil-free urban agriculture in space-constrained environments. Awarded a research grant from STEAM for its contribution to sustainable food technology.

C/C++ TensorFlow PyTorch OpenCV Reinforcement Learning Raspberry Pi Arduino ESP32 MQTT WebSocket FastAPI Flutter Gyanisum Livekit

What I built

  • Programmed the embedded control layer on ESP32 and Raspberry Pi to interface with pH, EC, temperature, and humidity sensors
  • Implemented sensor data acquisition and actuator control to manage pumps and nutrient dosing mechanisms via MQTT
  • Developed the backend infrastructure for the computer vision pipeline, enabling image uploads and AI model inference
  • Developed a FastAPI backend with MQTT and WebSocket support for real-time device monitoring and automation
  • Integrated LiveKit for real-time video streaming to enable live monitoring of plant growth
  • Built a Flutter mobile application for monitoring sensor data, live streams, and remote control
  • Deployed the system on a local edge server to support real-time processing and device communication
  • The project proposal was selected and funded by STEAM, supporting research in AI-powered sustainable agriculture

TrekSathi

Play Store
Play Store

A trekking companion designed to enhance the travel experience for tourists in Nepal. TrekSathi provides trail navigation, elevation profiles, weather forecasting and safety features for trekkers across Nepal's Himalayan routes. Published on Google Play under bala.ai, rated 3+.

Flutter Dart Google Maps API Django MongoDB OpenWeather API Firebase

What I built

  • Built the complete Flutter app from scratch — UI design, state management, navigation, and all screens
  • Integrated Google Maps SDK with custom trail overlays, waypoint markers, and elevation profile charts
  • Collected and structured raw trekking trail data into a reliable geospatial dataset
  • Connected OpenWeatherMap API for hyper-local mountain weather forecasts with severe weather alerts
  • Developed a Django + MongoDB backend with REST APIs for trail data, trip history, and trekker profiles
  • Implemented offline map caching for use without internet in remote mountain areas
  • Added agentic AI features for autonomous trek planning suggestions and personalized route recommendations
  • Managed the full Play Store release pipeline under the bala.ai account
04

Tiffin

Play Store
Play Store

A home-cooked meal delivery platform connecting students and office workers with local home chefs, hotels, and restaurants in Nepal. Tiffin promotes healthy eating, reduces food waste, and gives home cooks an income stream — all through a clean, fast Flutter app published under bala.ai. Each restaurant or hotel has its own cloud kitchen, allowing customers to pre-order meals before arrival.

Flutter Dart Firebase Firestore Node.js Google Maps

What I built

  • Designed and developed the full Flutter app for customers and home chefs
  • Built real-time order tracking using Firebase Firestore streams
  • Implemented Google Maps-based delivery radius matching customers to nearby chefs
  • Created a Node.js admin backend with APIs for user management, order history, and analytics
  • Handled full Play Store publishing under bala.ai
